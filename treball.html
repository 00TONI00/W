Durant el segon i tercer trimestre hem creat una pàgina web utilizant HTML per les estructures, CSS per l'estil visual i JavaScript per afegir interactivitat. El resultat és una pàgina web atractiva i funcional amb algunes animaciones, efectos de desplazamiento i algunes formules perquè es facin algunes comandes.

Amb el programa de codi obert Processing, hem creat una obra d'art interactiva utilitzant codi per crear formes, colors, animacions i interacció amb l'usuari. Hem interacció amb l'usuari. Hem experimentat amb diferents efectes visuals per crear una obra única i personalitzada.

Hem creat uns petits videojocs interactius sense utilitzar JavaScript, amb l'eina de programació de gràfics i animacions Processing. Hem utilitzat funcions i variables per gestionar la interacció dels usuaris, com la detecció de col·lisions i el moviment dels personatges, per crear jocs de plataformes i carreres divertits.
